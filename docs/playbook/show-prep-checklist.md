# Show Prep Checklist — HEAT4YAFEAT

Run this checklist for every show. It takes 15 minutes when you've done it 10 times.

---

## 48 Hours Before Show

### Inventory
- [ ] Finalize show lot — pull exact SKUs, confirm quantity
- [ ] Run StockX + GOAT comps on every item — document floor price
- [ ] Photograph any items not yet shot (box, pair, detail, auth tags)
- [ ] Organize inventory in show order (openers → mid-tier → anchors)
- [ ] Label or tag each item with lot # if running numbered format

### Promotion
- [ ] Post IG Story: "Going live [Day] at [Time] — here's a preview" (show 3–5 items)
- [ ] Post to Whatnot schedule (Upcoming Shows section on profile)
- [ ] Send push via Whatnot "Notify followers" feature (if available)
- [ ] Post TikTok sneak peek if active on platform
- [ ] Update eBay store banner/announcement with live date

---

## Day of Show

### Morning / Afternoon
- [ ] Pull all show inventory to your setup area
- [ ] Double-check every pair — missing laces? wrong size in box? fix now
- [ ] Charge all devices (phone/camera, tablet for chat monitor, backup battery)
- [ ] Test internet connection — run speedtest.net, confirm 10+ Mbps upload
- [ ] Eat — you're going to be on for 90 minutes minimum

### 2 Hours Before
- [ ] Clean and set up table/display surface
- [ ] Position lighting — check camera view, eliminate shadows on product
- [ ] Set up second screen or tablet at eye level for chat monitoring
- [ ] Do a solo camera test: check frame, lighting, audio levels, no lag

### 30 Minutes Before
- [ ] Open Whatnot app → Schedule Show → confirm all settings (title, category, start time)
- [ ] Write show title: include the highlight item + format — e.g. "HEAT4YAFEAT | Jordan 1s, Dunks & More | $1 Starts 🔥"
- [ ] Load thumbnail image if customizable
- [ ] Put phone on Do Not Disturb (keep only Whatnot notifications active)
- [ ] Have water bottle at the table — voice fatigue is real
- [ ] Lay out first 10 lots in order — don't hunt for items mid-show
- [ ] Brief any helpers on their role (box opener, chat moderator, shipping prep)

---

## Show Open (First 5 Minutes)

Run this exact sequence every time — it creates a recognizable ritual:

1. **Greeting:** "What's up everybody, welcome back to HEAT4YAFEAT — I'm [name], your sneaker plug. If you're new, hit that follow, you're in the right place."
2. **Rules:** "Everything tonight is fully authenticated, ships within 24 hours. No returns on live show items. Let's get into it."
3. **Preview tease:** Hold up or gesture to 2–3 anchor items. "We're closing with [grail] tonight — y'all are gonna want to stick around."
4. **First lot:** Open the room with a guaranteed crowd-pleaser under $75 at $1 start.

---

## During the Show

### Cadence Per Lot
- Display item to camera (rotate 360°, show box label, tongue tag, sole)
- Name it: "This is a [Year] [Colorway] in a size [size] — deadstock, never worn"
- Tell the story (30 seconds max): release context, why it matters, who it's for
- Start the bid
- Call out bidders by username while clock runs
- Sell, announce winner, set aside in "sold" pile

### Chat Management
- Greet every new follower by name, every show
- Answer authentication questions directly — never defensive, always educational
- Ignore or ban bad-faith lowballers/trolls on sight (don't engage)
- Designated moderator if you have help — they handle chat, you handle product

### Momentum Rules
- If a lot stalls with no bids at 30 seconds: narrate more, lower floor if needed, or move on and re-list later
- If you're on a hot streak: push through without long breaks — don't kill momentum with logistics talk
- Energy dips? Hold up an anchor item as a "coming soon" teaser. Room re-engages instantly.

---

## Show Close

- [ ] "Last lot of the night" announcement — make it the second-best anchor piece
- [ ] Final follow ask: "If you've been watching all night and haven't followed yet — do that now. It's free and it's how you know when we go live."
- [ ] Thank regulars by name
- [ ] Announce next show date and time
- [ ] End on energy, not exhaustion

---

## Within 24 Hours After Show

### Shipping
- [ ] Print all labels through Whatnot (or ShipStation if integrated)
- [ ] Pack items — tissue paper, thank-you card if using, box tape
- [ ] Drop off or schedule USPS pickup
- [ ] Mark items as shipped in Whatnot platform

### Admin
- [ ] Document show results: items listed, items sold, total GMV, sell-through %
- [ ] Note any items that didn't sell — adjust reserve price or re-list next show
- [ ] Screenshot final viewer count and follower gain for weekly KPI tracking
- [ ] Respond to any post-show buyer messages within 2 hours

### Content Repurpose
- [ ] Clip highlight moments from show (best lot reveals, big bids, reactions)
- [ ] Post best clip to TikTok/IG Reels: "This [shoe] just sold for $[price] 🔥 — we go live every [day]"
- [ ] Add show replay link to IG bio if available
