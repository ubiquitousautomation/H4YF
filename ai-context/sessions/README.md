# Session Logs

One file per session. Filename format: `YYYY-MM-DD-<agent>.md`

Examples:
- `2026-06-21-claude.md`
- `2026-06-22-gemini.md`

## Template

```markdown
# Session: YYYY-MM-DD — <Agent>

## Goal
What this session set out to do.

## Done
- Bullet list of completed actions

## Decisions Made
- Key choices and the reasoning

## Observations / Learnings
- Anything worth remembering (also add to memory.md)

## Left For Next Session
- What remains
```
